# Next Inn - Hotel Booking System

A modern hotel booking system built with HTML, CSS, JavaScript, and PHP.

## Features

- User registration and authentication
- Room browsing and booking
- User dashboard
- Responsive design
- Secure session management

## Requirements

- XAMPP (or similar PHP/MySQL environment)
- PHP 7.4 or higher
- MySQL 5.7 or higher

## Installation

1. Clone this repository
2. Place the project in your XAMPP `htdocs` directory
3. Start XAMPP and ensure MySQL is running
4. Import the `database.sql` file into your MySQL database
5. Update database credentials in `config/database.php` if needed
6. Access the application through `http://localhost/yoyo`

## Project Structure

```
yoyo/
├── api/              # PHP API endpoints
├── config/           # Configuration files
├── images/           # Image assets
├── *.html            # Frontend pages
├── style.css         # Stylesheet
├── script.js         # JavaScript functionality
└── database.sql      # Database schema
```

## Database

The database schema is provided in `database.sql`. Import it to set up the required tables.
